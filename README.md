# 群友识别 - AstrBot 插件

基于大语言模型的 QQ 群聊智能识别插件，自动建立群友身份映射，智能提取外号/别名，支持管理员交互修正。

## 功能特性

### 群友身份映射
- 实时追踪群友 QQ 号与当前昵称的对应关系
- 群友改名时自动同步更新
- JSON 持久化存储，支持本地缓存

### 智能外号提取
- 集成大语言模型分析群聊历史记录
- 自动识别群友的常用外号、别名和特征称呼
- 支持多轮迭代优化，动态更新外号数据库
- 可配置提取间隔，防止频繁调用 LLM

### 管理员交互修正
- 自然语言指令修正身份对应错误
- 权限验证，仅 AstrBot 管理员可操作
- 修正确认流程，防止误操作
- 修正历史记录，支持追溯与审计

### 数据安全
- 定时自动备份，可配置间隔和保留数量
- 手动备份/恢复功能
- 原子写入，数据损坏时自动回退

## 安装

将插件目录放入 AstrBot 的 `data/plugins/` 目录下：

```
AstrBot/
└── data/
    └── plugins/
        └── 群友识别/
            ├── main.py
            ├── metadata.yaml
            ├── _conf_schema.json
            ├── requirements.txt
            ├── data_manager.py
            ├── nickname_extractor.py
            └── admin_handler.py
```

在 AstrBot WebUI 的插件管理页面启用插件即可。

## 配置说明

在 WebUI 插件管理页面可直接配置以下参数：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| max_chat_history | 200 | LLM 分析时使用的最近聊天记录条数 |
| nickname_extraction_interval | 3600 | 两次外号提取的最小间隔（秒） |
| auto_extract_enabled | false | 是否启用自动外号提取 |
| auto_extract_threshold | 500 | 累积消息数达到后自动触发提取 |
| backup_interval | 3600 | 自动备份间隔（秒） |
| max_backup_count | 10 | 保留的最大备份数量 |
| correction_confirm_timeout | 60 | 管理员修正确认超时（秒） |
| llm_provider_id_override | "" | 指定 LLM 提供商 ID，留空使用默认 |
| log_level | INFO | 日志级别 |

## 命令列表

### 基础查询（所有人可用）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/qy_help` | 查看帮助信息 | `/qy_help` |
| `/qy_info [@某人\|QQ号]` | 查看群友信息和外号 | `/qy_info @小明` 或 `/qy_info 123456789` |
| `/qy_list [页码]` | 分页列出本群群友 | `/qy_list` 或 `/qy_list 2` |
| `/qy_search <关键词>` | 搜索群友 | `/qy_search 老王` |
| `/qy_stat` | 查看插件统计和外号排行 | `/qy_stat` |

### 外号提取（管理员）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/qy_extract` | 手动触发 LLM 外号提取 | `/qy_extract` |
| `/qy_extract <QQ号>` | 提取指定群友的外号 | `/qy_extract 123456789` |

### 管理员修正（管理员）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/qy_admin` | 查看管理员帮助 | `/qy_admin` |
| `/qy_admin 修正 <QQ号> <新昵称>` | 修改群友昵称映射 | `/qy_admin 修正 123456789 新名字` |
| `/qy_admin 添加外号 <QQ号> <外号>` | 手动添加外号 | `/qy_admin 添加外号 123456789 老王` |
| `/qy_admin 删除外号 <QQ号> <外号>` | 删除外号 | `/qy_admin 删除外号 123456789 老王` |
| `/qy_admin 查找 <关键词>` | 搜索群友 | `/qy_admin 查找 小明` |
| `/qy_admin 修正记录` | 查看修正历史 | `/qy_admin 修正记录` |
| `/qy_admin 备份列表` | 查看备份文件 | `/qy_admin 备份列表` |
| `/qy_admin 恢复备份 <文件名>` | 从备份恢复 | `/qy_admin 恢复备份 backup_20260101.zip` |

### 系统管理（管理员）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/qy_backup` | 手动备份数据 | `/qy_backup` |

> **注意**：所有管理员命令在执行危险操作（修改/删除）时会要求二次确认，回复「确认」执行，「取消」放弃。

## 数据存储

插件数据存储在 `data/` 子目录下：

```
群友识别/
└── data/
    ├── members.json          # 群友身份映射
    ├── nicknames.json        # 外号数据库
    ├── chat_cache.json       # 聊天记录缓存
    ├── correction_log.json   # 修正审计日志
    └── backups/              # 数据备份
        └── backup_YYYYMMDD_HHMMSS.zip
```

## 工作流程

```
群聊消息
    │
    ▼
┌─────────────────────┐
│  消息监听 (regex)    │ ← 每条消息都经过这里
├─────────────────────┤
│ 1. 更新身份映射      │   记录 QQ号 → 昵称
│ 2. 缓存聊天记录      │   截取最近 N 条消息
└────────┬────────────┘
         │
    ┌────▼────┐
    │ 命令路由 │
    └────┬────┘
         │
    ┌────┴────────────┐
    │                 │
    ▼                 ▼
┌─────────┐    ┌──────────────┐
│ 查询命令 │    │ /qy_extract  │
│ qy_info │    │ (管理员触发)  │
│ qy_list │    └──────┬───────┘
│ qy_stat │           │
└─────────┘           ▼
              ┌──────────────┐
              │ LLM 外号提取  │
              │ 分析聊天记录  │
              │ 返回 JSON     │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │ 更新外号数据库 │
              └──────────────┘
```
